# AmoTool

**amoCRM — Задачи + SMS**

Десктопное приложение для Windows: выгружает задачи из amoCRM и отправляет массовые SMS мастерам и клиентам.

## Возможности

- Авторизация в amoCRM через Selenium
- Просмотр и фильтрация задач
- Массовая отправка SMS через GSM-шлюз
- Шаблоны сообщений
- Автообновление через GitHub Releases
- Хранение учётных данных (Windows DPAPI)

## Технологии

Python 3, Selenium, BeautifulSoup, tkinter, PyInstaller.
